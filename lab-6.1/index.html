Bar Chart with Mouse Over

D3 Interactive Chart Demo

Mouse Over Effects: Hover over any bar to see it highlight in orange and display a custom SVG tooltip showing the exact value.

Add/Remove: Use the buttons to dynamically add or remove bars - all new bars will also have the interactive mouse over effects.

This demonstrates D3's powerful event handling capabilities for creating engaging, interactive data visualizations!
