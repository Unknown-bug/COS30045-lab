Methodology & Data Sources

This analysis is based on the Australian Road Deaths Database (ARDD) maintained by the Bureau of Infrastructure, Transport and Regional Economics (BITRE). The dataset contains information about road transport crash fatalities in Australia from 1989 to January 2020, providing comprehensive demographic and crash details.

Data Quality: Missing or unknown values are represented as -9 in the original dataset and have been handled appropriately in this analysis.

Visualization Tools: This interactive dashboard uses D3.js and Chart.js for data visualization, providing dynamic and responsive charts.
